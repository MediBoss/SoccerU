## Endpoints

All HTTP methods are supported.

|       Verb          |      Endpoint        |       Description              |
| -------------       |:--------------------:| ------------------------------:|
| GET                 | /teams               | Get all the teams available    |
| GET                 | /teams/:teamID       | Get data about a specific team |
